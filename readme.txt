== Description ==

DinhKore Plugin là một plugin WordPress chuyên dụng được thiết kế để bổ sung các Widget tùy chỉnh cho trình dựng trang Elementor.

Plugin này được xây dựng dựa trên cấu trúc chuẩn (Standard Architecture), giúp mã nguồn gọn gàng, dễ bảo trì và mở rộng. Đây là sản phẩm phục vụ cho việc học tập và triển khai các dự án thực tế của DinhKore.

**Các tính năng chính:**
* Cung cấp các Widget Elementor chuyên nghiệp (Ví dụ: Hello Widget).
* Cấu trúc code tổ chức theo module.
* Tương thích hoàn toàn với Elementor Free và Pro.
* Dễ dàng tùy biến (Customizable).

== Installation ==

1.  Tải thư mục `dinhkore-plugin` vào thư mục `/wp-content/plugins/` của website.
2.  Truy cập menu **Plugins** trong trang quản trị WordPress.
3.  Tìm **DinhKore Plugin** và nhấn **Kích hoạt (Activate)**.
4.  Mở trình sửa trang Elementor, tìm từ khóa "DinhKore" để sử dụng widget.

== Frequently Asked Questions ==

= Plugin này có miễn phí không? =
Có, DinhKore Plugin hoàn toàn miễn phí và mã nguồn mở (GPLv2).

Có, plugin này hoạt động dựa trên Elementor, nên bạn cần cài đặt và kích hoạt Elementor trước.
